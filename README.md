# Arrays Dinámicos, Estáticos y Herencia

Este repositorio contiene una colección de ejercicios prácticos en **Java** correspondientes a los temas 6 y 7 de programación. El objetivo es profundizar en la Programación Orientada a Objetos (POO) y el manejo de estructuras de datos lineales.

---

## 📂 Estructura del Repositorio

El proyecto está organizado en carpetas temáticas para facilitar la navegación:

### 📗 Tema 6: Clases, Herencia y Utilidades
En esta sección se exploran los pilares de la POO y el uso de librerías estándar:
* **`abstract` e `interface`**: Uso de clases abstractas e interfaces para definir contratos.
* **`herencia`**: Jerarquías de clases y reutilización de código.
* **`final`**: Uso del modificador para constantes y métodos/clases no heredables.
* **`String` y `string-format`**: Manipulación avanzada de cadenas de texto.
* **`java.time`**: Gestión moderna de fechas y horas.
* **`Ejercicios_abst_inter_final`**: Prácticas integradoras de estos conceptos.

### 📘 Tema 7: Arrays (Estructuras de Datos)
Dedicado al almacenamiento organizado de información:
* **`inicioArray`**: Fundamentos y sintaxis de los arrays.
* **`ejercicios_array_estatico`**: Manipulación de arreglos de tamaño fijo.
* **`arrays_dinamicos`**: Uso de estructuras flexibles (ArrayList) para datos de tamaño variable.

---
